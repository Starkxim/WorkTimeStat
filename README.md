# WorkTimeStat

WorkTimeStat 是一个用 Python 编写的项目，用于统计和分析工作时间。该项目旨在帮助用户更好地管理和优化他们的工作时间，提高工作效率。

## 功能

- 记录工作时间
- 生成工作时间报告
- 分析工作时间数据

## 安装

1. 克隆仓库：
    ```bash
    git clone https://github.com/yourusername/WorkTimeStat.git
    ```
2. 进入项目目录：
    ```bash
    cd WorkTimeStat
    ```
3. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```

## 使用

运行以下命令开始记录工作时间：
```bash
python main.py
```
